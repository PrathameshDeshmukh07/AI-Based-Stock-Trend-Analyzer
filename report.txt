AI-Based Stock Trend Analyzer — Probabilistic Learning on Time-Series Data


================================================================================
                        PROBLEM STATEMENT
================================================================================

Stock market prediction is a challenging task due to the inherently volatile
and non-linear nature of financial data. Investors and traders require
data-driven tools that can analyze historical price movements, identify
patterns, and generate reliable forecasts to make informed decisions.

The problem is to develop an AI-based system that can:
  1. Fetch and process real-time historical stock price data.
  2. Apply probabilistic learning techniques on time-series data to predict
     future stock price trends.
  3. Compute standard technical indicators (RSI, MACD, Bollinger Bands, Moving
     Averages) to support analysis.
  4. Provide confidence intervals and probability scores to quantify the
     reliability of predictions.
  5. Present all results through an interactive, user-friendly web dashboard.


================================================================================
                            OBJECTIVE
================================================================================

The primary objectives of this project are:

  1. To apply AI-based probabilistic learning techniques (ARIMA and Exponential
     Smoothing) for stock trend prediction using historical time-series data.

  2. To build an ensemble forecasting model that combines ARIMA(5,1,2) and
     Damped Exponential Smoothing with a weighted average (60:40) for more
     robust predictions.

  3. To compute and visualize essential technical indicators — RSI (Relative
     Strength Index), MACD (Moving Average Convergence Divergence), Bollinger
     Bands, SMA (Simple Moving Average), and EMA (Exponential Moving Average).

  4. To generate probabilistic forecasts with 80% and 95% confidence intervals,
     along with probability of upward/downward price movement.

  5. To develop a full-stack web application with a Flask backend and a modern
     glassmorphism frontend (using Chart.js) that provides an interactive
     dashboard for stock analysis.

  6. To classify the predicted trend as BULLISH, BEARISH, or NEUTRAL with a
     quantified strength score.

2. AI TECHNIQUES USED
---------------------

   a) ARIMA (AutoRegressive Integrated Moving Average) — Order (5,1,2):
      - A classical time-series forecasting model.
      - AR(5): Uses the last 5 lagged values for autoregression.
      - I(1): First-order differencing to make the series stationary.
      - MA(2): Uses the last 2 forecast error terms for moving average.
      - Generates point forecasts with statistical confidence intervals
        at both 80% (alpha=0.20) and 95% (alpha=0.05) levels.

   b) Exponential Smoothing (Holt's Damped Trend):
      - Captures the overall trend in the data with a damping factor.
      - Uses additive trend component with damping to prevent over-
        extrapolation of the trend into the future.
      - Parameters are optimized automatically during model fitting.

   c) Ensemble Method:
      - The final forecast is a weighted combination:
            Forecast = 0.6 × ARIMA + 0.4 × Exponential Smoothing
      - This ensemble approach reduces model-specific biases and improves
        prediction robustness.

   d) Probability Estimation:
      - Uses the last 60 days of closing price returns.
      - Calculates the proportion of days with positive returns as the
        probability of upward movement.
      - Probability of downward movement = 1 - Probability of upward movement.


3. TECHNICAL INDICATORS COMPUTED
---------------------------------

   a) RSI (Relative Strength Index) — 14-period:
      - Measures momentum of price changes.
      - RSI > 70: Overbought (potential sell signal).
      - RSI < 30: Oversold (potential buy signal).
      - Formula: RSI = 100 - (100 / (1 + RS)), where RS = Avg Gain / Avg Loss.

   b) MACD (Moving Average Convergence Divergence):
      - MACD Line = EMA(12) - EMA(26)
      - Signal Line = EMA(9) of MACD Line
      - Histogram = MACD Line - Signal Line
      - Bullish when MACD crosses above Signal; Bearish when below.

   c) Bollinger Bands (20-period, 2 standard deviations):
      - Middle Band = SMA(20)
      - Upper Band = SMA(20) + 2 × Standard Deviation(20)
      - Lower Band = SMA(20) - 2 × Standard Deviation(20)
      - Used to identify overbought/oversold conditions and volatility.

   d) Moving Averages:
      - SMA(20) — Short-term Simple Moving Average
      - SMA(50) — Medium-term Simple Moving Average
      - EMA(12) — Short-term Exponential Moving Average
      - EMA(26) — Medium-term Exponential Moving Average

   e) Volume Moving Average (20-period):
      - Helps identify unusual trading volume spikes.

7. AI PIPELINE (WORKFLOW)
--------------------------

   Step 1: User enters a stock symbol (e.g., AAPL, RELIANCE.NS).

   Step 2: Backend fetches historical OHLCV data from Yahoo Finance API
           for the selected time period (default: 1 year).

   Step 3: Data is processed and split into two parallel pipelines:

            Pipeline A — Technical Indicators:
            Compute RSI(14), MACD(12,26,9), Bollinger Bands(20,2), SMA(20,50),
            EMA(12,26), and Volume MA(20).

            Pipeline B — AI Prediction:
            ┌─────────────────────────────────────────┐
            │         Historical Closing Prices        │
            │                    │                     │
            │       ┌────────────┴────────────┐        │
            │       │                         │        │
            │   ARIMA(5,1,2)       Exponential Smoothing│
            │       │               (Damped Trend)     │
            │       └────────────┬────────────┘        │
            │                    │                     │
            │       Ensemble (60% ARIMA + 40% ES)      │
            │                    │                     │
            │    Forecast + 80%/95% Confidence Bands   │
            │                    │                     │
            │    Trend: BULLISH / BEARISH / NEUTRAL    │
            │    Probability of Up/Down Movement       │
            └─────────────────────────────────────────┘

   Step 4: All results are returned via REST API as JSON responses.

   Step 5: Frontend renders interactive charts and dashboard cards using
           Chart.js, displaying prices, indicators, forecasts, and
           confidence bands.


================================================================================
                        PROPOSED SOLUTION
================================================================================

The proposed solution is a full-stack AI-powered web application that combines
probabilistic learning with time-series analysis for stock trend prediction.

1. Data Acquisition:
   - Use the yfinance library to fetch real-time historical stock data from
     Yahoo Finance, supporting global stock exchanges.

2. AI Forecasting Engine:
   - Implement an ensemble model combining:
     a) ARIMA(5,1,2) — Captures autoregressive patterns and trends in price data.
     b) Exponential Smoothing with Damped Trend — Captures overall trend direction
        while limiting over-extrapolation.
   - The ensemble uses a 60:40 weighted average for balanced prediction.
   - Generate 80% and 95% confidence intervals for uncertainty quantification.
   - Compute probability of price going up or down using recent return distributions.

3. Technical Analysis Module:
   - Compute industry-standard indicators (RSI, MACD, Bollinger Bands, SMA/EMA)
     to provide a holistic view of stock health alongside AI predictions.

4. Web Application:
   - Backend: Flask REST API serving stock data, predictions, and indicators as
     JSON endpoints.
   - Frontend: Responsive HTML/CSS/JS dashboard with Chart.js for interactive
     visualization.
   - AI Chatbot: Dynamic conversational interface for querying stock data.

5. Trend Classification:
   - Based on predicted percentage change over 30 days:
     * Change > +2%  → BULLISH
     * Change < -2%  → BEARISH
     * -2% to +2%    → NEUTRAL
   - Strength score calculated as: min(|change%| / 10, 1.0)
